# Commands

# Add card to wishlist `.wa`
Single card:
```
.wa One Punch Man || Saitama
``` 

Multiple cards:
```
.wa One Piece || Monkey D. Luffy, Roronoa Zoro
```

# Remove from wishlist `.wr`
Single card:
```
.wr One Piece || Monkey D. Luffy
``` 

Multiple cards:
```
.wr One Piece || Monkey D. Luffy, Roronoa Zoro
```
